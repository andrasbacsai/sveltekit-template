# SvelteKit template

# About
https://andrasbacsai.com/ssr-authentication-guide-for-sveltekit

# Includes
- TailwindCSS (canary)
- Node adapter
- Guarded endpoints / pages

# Usage

## Development
`npm run dev` (or `pnpm dev`)

## Build
`npm run build` (or `pnpm build`)

## Production
`npm run start` (or `pnpm start`)
